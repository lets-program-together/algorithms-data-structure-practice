## Algorithms: Searching

### Linear Search
| Best Case | Worst Case |
|-----------|------------|
|    O(1)   |    O(n)    |


### Binary Search
*Requirement: data is sorted*

| Best Case | Worst Case |
|-----------|------------|
|    O(1)   |  O(log n)  |

## Algorithms: (Graph / Tree) Traversal

### Breadth-First Search
* Shortest Path
* Closer Nodes
* Requires more resources in-memory.

### Depth-First Search
* Less Memory
* Does Path Exist
* Can be slow
